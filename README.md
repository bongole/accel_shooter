# Accel Shooter
This is my toy project for learning Go language.

## BUILD & LAUNCH SERVER
### OSX
```sh
$ brew install imagemagick
$ cd accel_shooter/
$ go build
$ ./accel_shooter
```

### CentOS
```sh
$ sudo yum install -y ImageMagick-devel
$ cd accel_shooter/
$ go build
$ ./accel_shooter
```

Open http://localhost:8080/ in your browser.
